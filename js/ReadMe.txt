=== Javascript Sources for Leaflet Maps ===

Most of these files are imported from other repositories to provide handling for Leaflet maps.
WordPress no longer allows plugins to pull in 3rd party scripts from other servers,
so these are local copies of publicly available scripts.

== Source URLs ==

* https://unpkg.com/leaflet@1.9.4/dist/leaflet.js
* https://unpkg.com/@raruto/leaflet-gesture-handling@1.3.5/dist/leaflet-gesture-handling.min.js
* https://github.com/brunob/leaflet.fullscreen/releases/tag/v3.0.1
* https://cdnjs.cloudflare.com/ajax/libs/proj4js/2.5.0/proj4.js
* https://cdnjs.cloudflare.com/ajax/libs/proj4leaflet/1.0.2/proj4leaflet.min.js
* https://api.tiles.mapbox.com/mapbox.js/plugins/leaflet-omnivore/v0.3.1/leaflet-omnivore.min.js
* https://cdnjs.cloudflare.com/ajax/libs/leaflet-gpx/1.5.0/gpx.min.js
* https://unpkg.com/@raruto/leaflet-elevation@1.7.0/dist/leaflet-elevation.min.js
* https://github.com/skirridsystems/leaflet.browser.print
* https://cdn.jsdelivr.net/npm/leaflet.locatecontrol@0.76.1/dist/L.Control.Locate.min.js
